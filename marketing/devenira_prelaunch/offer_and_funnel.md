# Offer And Funnel

## Single Prelaunch Offer
`Join the first 100 beta users and start with one free body check-in.`

This is the only offer the landing page should push.

Do not mix it with:
- upgrade language
- pricing-first language
- full product-tour language
- calorie-tracker positioning

## Single CTA
`Join the beta`

Secondary CTA:
`See how the beta works`

## CTA Destination
Primary destination:

`/try?source=landing_prelaunch`

Reason:
- the product already has a working guest body-check flow
- a real action is stronger than a passive waitlist
- this makes prelaunch feel alive without pretending the full app is finished

## Funnel Logic

### Entry
Traffic lands on the homepage.

### First Decision
User chooses one of two paths:
- `Join the beta`
- `See how it works`

### Beta Action
The beta action leads to the guest try flow where the user:
- uploads a photo
- sees a body fat estimate range
- understands the weekly loop
- is invited to create an account for weekly tracking

### Conversion Event
The real conversion is not only guest scan completion.

Primary conversion:
- click into beta flow

Secondary conversions:
- guest scan completed
- account created after result

## CTA Copy By Section

### Hero
- primary: `Join the beta`
- secondary: `See the 3-step loop`

### Founder Story
- `Join the beta built for the hard middle`

### Goal / Start Section
- `Join beta for this goal`

### Final CTA
- `Join the first 100 beta users`

## Source Naming
Use source tags so traffic can be read later.

Recommended values:
- `landing_prelaunch`
- `landing_founder_story`
- `landing_goal_cut`
- `landing_goal_recomp`
- `landing_goal_muscle`
- `landing_final_cta`
- `footer_join_beta`

## What Success Looks Like
If someone asks what they are signing up for, the answer should be:

`You are getting early access to a weekly AI body check-in flow. Start with one scan now, then create an account if you want to keep the weekly proof loop going.`

## Offer Rules
- one page, one main CTA
- one offer, repeated often
- no pricing block on the main landing during prelaunch
- no fake scarcity beyond the beta framing
- no hard claim that the product is fully released
